# Review App

A Python application for managing review queues and processing review data.

## Files
- `review_app_json.py` - Main application logic
- `human_review_queue.json` - Queue data
- `reviewed_output.json` - Processed review results
- `run_app.bat` - Windows batch script to run the application

## Usage
Run the application using:
```
run_app.bat
```
